   ```

   ```

   - Task specification (what to do)
   - Context block (what the agent needs to know — paths, prior outputs)
   - Required reading block (if the agent must read specific files)
   - Acceptance criteria (how the orchestrator will know the agent succeeded)
   - Output format (structured — tables, markdown sections, completion marker)
   - Constraints (what NOT to do — e.g., "do not modify files outside `.design/`")

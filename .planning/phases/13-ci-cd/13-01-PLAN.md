# Plan 13-01: Extend `ci.yml` — Job Separation + Fast-Fail Ordering

**Phase:** 13 — CI/CD
**Wave:** A (parallel with 13-02, 13-03)
**Requirements:** CICD-01 (cross-platform matrix preserved), CICD-02 (fast-fail ordering via `needs:`)
**Depends on:** Phase 12 `ci.yml` (matrix already in place)

## Objective

Restructure `.github/workflows/ci.yml` from a single `test` job into a named job skeleton — **lint → validate → test → security → size-budget** — with `needs:` edges enforcing fast-fail ordering. Preserve the existing matrix verbatim. This plan lands the *shell*; later plans (13-02..13-05) fill each job's `steps:` with real tools.

**Purpose:** every CI run should burn cheap lint/validate cycles before spinning up six matrix machines. All 13-0x plans attach to the same workflow file, so the structure must be right before content lands.

**Output:** a single modified `ci.yml` with 5 named jobs, `needs:` wiring, and stub steps that echo a marker string so each job is observable in the Actions tab even before content lands.

name: CI

on:
  push:
    branches: ["**"]
  pull_request:
    branches: ["**"]

jobs:
  lint:
    name: "Lint (markdown + frontmatter + stale-refs)"
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Setup Node.js 22
        uses: actions/setup-node@v4
        with:
          node-version: 22
      - name: Placeholder — filled by plan 13-02
        run: echo "lint job placeholder — plan 13-02 lands markdownlint + lychee + frontmatter + stale-refs"

  validate:
    name: "Validate (schemas + plugin + shellcheck)"
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Setup Node.js 22
        uses: actions/setup-node@v4
        with:
          node-version: 22
      - name: Placeholder — filled by plan 13-02 + 13-03
        run: echo "validate job placeholder — plan 13-02 lands ajv schemas; plan 13-03 lands plugin validate + shellcheck"

  test:
    name: "Test (Node ${{ matrix.node-version }} / ${{ matrix.os }})"
    needs: [lint, validate]
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        node-version: [22, 24]
        os: [ubuntu-latest, macos-latest, windows-latest]
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Setup Node.js ${{ matrix.node-version }}
        uses: actions/setup-node@v4
        with:
          node-version: ${{ matrix.node-version }}
      - name: Run tests
        run: npm test

  security:
    name: "Security (secrets + injection scan)"
    needs: [test]
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Placeholder — filled by plan 13-04
        run: echo "security job placeholder — plan 13-04 lands gitleaks + injection-scanner"

  size-budget:

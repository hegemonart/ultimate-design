### No-python3 Degraded Path

Running the hook under a PATH missing python3 (`/bin:/usr/bin/nonexistent`) exits rc=0. Network fetch is also blocked (no curl in /bin on macOS), so no cache is written during the test — the relevant guarantee (silent exit 0 when python3 absent) holds. If network + curl had succeeded, `changelog_excerpt` would be empty per `extract_body` returning 0 on missing python3.
